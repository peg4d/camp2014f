# Krile StarryEyes

Krile StarryEyes is a twitter client, but this is not just a twitter client.

## Target Platform

Windows Vista or later(we recommend Windows 8.1 x64), and .NET Framework 4.5 .

Visit [krile.starwing.net](http://krile.starwing.net/) .
